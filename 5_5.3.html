Chapter 5 Targets

  • Draw quadratic graphs.
  • State the properties of quadratic graphs (positive/negative coefficient, maximum/minimum points, symmetry).

Reference: Properties of Quadratic Graphs (y = ax² + bx + c)

Shape of the Curve (Parabola):
If a > 0 (positive): U-shape (has a Minimum point).
If a < 0 (negative): N-shape (has a Maximum point).
Key Features:
Line of Symmetry: A vertical line passing exactly through the turning point.
y-intercept: The curve cuts the y-axis at (0, c).
