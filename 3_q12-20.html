Factorisation by Grouping:
ax + ay + bx + by = a(x + y) + b(x + y) = (x + y)(a + b)
Substitution Trick:
If x² - 5x + 6 = (x-2)(x-3), then
(y+1)² - 5(y+1) + 6 = (y+1-2)(y+1-3).
